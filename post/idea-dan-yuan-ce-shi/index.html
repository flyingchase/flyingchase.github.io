# idea 单元测试

2021-08-30

安装插件

配置 test 目录

  • 在 src 同级生成 test 目录

  • 右键 test 目录设为 test resourceroot

生成测试类

  • 在待测试的类或者方法上使用快捷键 cmd+shift+t

  • 选择 Junit5 + 待测方法名

  • 自动上设置的test 目录新建对应 package 下的测试类

编写测试代码

  • 使用 new 创建原先的代码类并传入方法
  • 使用assertEquals等进行测试
